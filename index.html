Dice Game

In the first year of the bachelor's degree, for the multidisciplinary project, we had to create a set of two dice with two parts: an electronic part and a computer program. The two parts were linked by a communication card. There were six of us in the group, each with a task to perform. The project ran throughout the second quarter.

Shape detector

This project consisted of two technologies: Arduino for embedded systems and the Raspberry for the web and database.

The idea for the project was to detect different shapes placed on bricks. The bricks move along a conveyor belt.

To carry out the project, we had to use an Arduino which contained a source code for object detection. Various sensors were used to make this possible : IR (Infrared) obstacle sensor and a camera equipped with AI (artificial intelligence). Every sensor was plugged into the Arduino.

A raspberry was used to host a website, a database and a software called Nodered which enables raspberry and Arduino to communicate information to each other.
