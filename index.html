Shape detector

In 2023, the students studying for a second bachelor's degree in industrial IT had to make a project for the multidisciplinary project course. The idea of the project was to detect different shapes placed on bricks that were moving along a conveyor belt.

This project included four disciplines: technical English for the presentation, embedded systems for the sensor, database to stock the data, and web 2 to display all the data.

To carry out the project, we had to use an Arduino which contained a source code for object detection. Various sensors were used to make this possible: an IR (Infrared) obstacle sensor and a camera equipped with AI (artificial intelligence). Every sensor was plugged into the Arduino. A Raspberry was used to host a website, a database, and a software called Nodered which enabled Raspberry and Arduino to communicate with each other.

We worked in a team for this project and I was with Loic and Steven. We already did the first-year project together, so this project was not the first time we worked together. Thanks to that, we knew how to work together.

I learned a lot of different skills during the making of this project. I have improved my ability to work in a group. I’m now pretty good at making websites and databases. It was my main weakness regarding the project before making it.
